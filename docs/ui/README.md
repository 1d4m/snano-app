## 画面一覧

| No  | 画面名称                 | URL                     | 説明                                 |
| --- | ------------------------ | ----------------------- | ------------------------------------ |
| 1   | ログイン画面             | `/login`                | ログインを行う画面                   |
| 2   | プレイリスト一覧         | `/playlists`            | プレイリストを一覧表示する           |
| 3   | プレイリスト詳細         | `/playlists/[id]`       | 特定のプレイリストの詳細を表示する   |
| 4   | プレイリストアイテム詳細 | `/playlists-items/[id]` | プレイリストアイテムの詳細を表示する |
| 5   | アクティビティ           | `/activity`             | ユーザーの活動履歴を表示する         |
| 6   | アカウント設定           | `/account`              | アカウントに関する設定を行う         |
